# Echo Chat App 🤖

A modern AI-powered chatbot application built with React and Tailwind CSS that allows users to interact with Google's Gemini AI through a clean, responsive, and user-friendly chat interface.

> ⚠️ This project is frontend-focused and uses the Gemini API to generate AI-powered responses in real time.

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm

### Installation

```bash
# Clone the repository
git clone https://github.com/CodeWithNouman99/Echo-chat-app.git

# Navigate to the project folder
cd Echo-chat-app

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open:

```text
http://localhost:5173
```

in your browser.
